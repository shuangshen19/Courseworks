
 ***************
 ** 檔案說明: **
 ***************
	1. CardSet: 儲存牌桌資訊，包含 52 張撲克牌的位置，以 13*4 的座標方式表示，卡牌之間以逗號隔開
	
	2. FileIO.c: File I/O 的 Example code。可直接使用亦可改寫，但務必確保作業程式正常運作。

		◎ 範例程式變數說明:
			
			fp : 讀取檔案
			
			filename：欲讀取的檔案名稱

			token：暫存字串切割結果

			search：儲存欲拿來當作切割判斷的字元

			result：二維陣列儲存 52 張卡牌的內容
